## Brand Colors

<SystemUIShowcases

/>

## Gray Colors

There has been [research](https://uxmovement.com/content/why-you-should-never-use-pure-black-for-text-or-backgrounds/) [done](https://uxplanet.org/basicdesign-never-use-pure-black-in-typography-36138a3327a6) showing that pure black can cause eye-strain. This is why we've chosen not to include a `100` value.

<SystemUIShowcases
  theme={{ colors: theme.color.gray }}
  showcaseKey="bg"
  componentProps={square}
/>

## Variant Colors

Variant colors are generated using [`coloralgorithim`](https://github.com/k-vyn/coloralgorithm) and helps produce much cleaner color pallettes.

<React.Fragment>
  {Object.keys(theme.color.variant).map((vars)=>{
    return (<React.Fragment>
    <h3>{vars}</h3>
<SystemUIShowcases
  theme={{ colors: theme.color.variant[vars] }}
  showcaseKey="bg"
  componentProps={square}
/>
</React.Fragment>)
  })}

</React.Fragment>
